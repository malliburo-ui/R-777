# Проекты (левая колонка)

Картинки меняются, когда пользователь **наводит курсор на левую зону** и **крутит колёсико** (на телефоне — свайп).

---

## Способ 1 — самый простой (рекомендуется)

1. Положи картинку в папку **`images/`**  
   Имя с номером в начале — так задаётся порядок:

   ```
   images/
     01-fintech.png
     02-branding.jpg
     03-poster.webp
   ```

2. В терминале (в папке проекта):

   ```bash
   npm run projects:sync
   ```

3. Обнови страницу в браузере.

Скрипт сам создаст **`projects.json`**.  
Название проекта берётся из имени файла: `02-branding.jpg` → **Branding**.

### Свои названия (необязательно)

Отредактируй **`projects.meta.json`**:

```json
{
  "03-poster.webp": {
    "title": "Poster series 2024"
  }
}
```

Потом снова: `npm run projects:sync`

---

## Способ 2 — вручную

1. Картинка в **`images/`**
2. Добавь объект в **`projects.json`**:

```json
{
  "id": "branding",
  "title": "Branding",
  "image": "03-branding.png"
}
```

3. Обнови страницу.

---

## Советы

| Что | Рекомендация |
|-----|----------------|
| Формат | PNG, JPG, WebP |
| Ширина | от 1200px |
| Имя файла | латиница, без пробелов: `04-my-work.png` |
| Порядок | префикс `01-`, `02-`, … |

---

## Где что лежит

```
public/projects/
  images/           ← сюда кидаешь картинки
  projects.json     ← список для сайта (auto или вручную)
  projects.meta.json ← свои названия (опционально)
```
